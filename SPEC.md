# 工具集合网站 - 项目规格说明

## 1. 概念与愿景

一个快速、简洁、实用的在线工具集合网站，采用经典的单页应用结构，每个工具独立模块。网站追求"秒开"体验，去除一切不必要的设计元素，让用户专注于工具本身。整体风格专业、干净、可信赖，让用户觉得这个网站可以长期使用。

## 2. 设计语言

### 美学方向
- 参考：简洁工具站的极简风格（如TinyPng、ConvertIO等工具站）
- 大面积留白，重点突出工具功能
- 卡片式工具展示，便于用户快速识别和选择

### 色彩方案
- 主色：#2563EB（蓝色，代表专业、可信赖）
- 次色：#1E40AF（深蓝，hover状态）
- 强调色：#10B981（绿色，用于成功状态）
- 警告色：#F59E0B（橙色，用于提示）
- 错误色：#EF4444（红色）
- 背景色：#F8FAFC（淡灰蓝）
- 卡片背景：#FFFFFF
- 文字主色：#1E293B（深灰）
- 文字次色：#64748B（中等灰）

### 字体
- 主字体：-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
- 代码字体："SFMono-Regular", Consolas, "Liberation Mono", Menlo, monospace

### 间距系统
- 基础单位：4px
- 常用间距：8px, 12px, 16px, 24px, 32px, 48px
- 卡片圆角：12px
- 按钮圆角：8px

### 动效哲学
- 所有过渡：150ms ease
- 按钮hover：轻微上浮 + 阴影加深
- 工具切换：淡入淡出，200ms
- 加载状态：简单的旋转动画

## 3. 布局与结构

### 页面结构
```
┌─────────────────────────────────────┐
│           顶部导航栏                 │
│   Logo + 工具分类导航 + SEO keywords │
├─────────────────────────────────────┤
│           英雄区域                   │
│    标题 + 简介 + 搜索/快速工具入口    │
├─────────────────────────────────────┤
│           工具网格                   │
│    分类展示所有可用工具卡片           │
├─────────────────────────────────────┤
│        当前激活工具区域              │
│    根据选择显示对应的工具界面         │
├─────────────────────────────────────┤
```

### 响应式策略
- 桌面端（>1024px）：3-4列工具网格
- 平板端（768-1024px）：2列工具网格
- 移动端（<768px）：单列布局，汉堡菜单

## 4. 功能与交互

### 核心工具列表

#### 4.1 图片格式转换
- 支持格式：JPG, PNG, WebP, GIF, BMP 互转
- 交互：拖拽上传或点击选择，实时预览，选择目标格式，一键转换
- 输出：点击下载转换后的图片
- 限制：单文件最大 10MB

#### 4.2 MD5加密
- 支持文本输入
- 输出：32位MD5哈希值
- 功能：一键复制结果

#### 4.3 简繁体转换
- 双向转换：简体 ↔ 繁体
- 实时转换，无需提交
- 支持大段文本

#### 4.4 二维码生成
- 输入：文本或URL
- 输出：可下载的PNG格式二维码
- 可自定义尺寸：100px - 400px

#### 4.5 Base64编码/解码
- 双向编码解码
- 支持文本和文件
- 实时转换

#### 4.6 URL编码/解码
- 双向转换
- 实时转换
- 一键复制结果

#### 4.7 颜色格式转换
- 输入格式：HEX, RGB, HSL
- 输出：三种格式都显示
- 实时预览颜色

#### 4.8 时间戳转换
- 支持：Unix时间戳 ↔ 可读日期
- 实时转换
- 显示当前时间戳

### 通用交互
- 工具切换：点击工具卡片，展开对应工具界面
- 错误提示：红色边框 + 错误信息
- 成功提示：绿色toast提示，2秒自动消失
- 空状态：灰色提示文字
- 加载状态：按钮显示加载动画，禁止重复点击

## 5. 组件清单

### 导航栏
- 默认：白色背景，轻微阴影
- 滚动时：阴影加深
- Logo：点击返回首页

### 工具卡片
- 默认：白色背景，圆角，轻微阴影
- Hover：阴影加深，轻微上浮
- Active/选中：蓝色边框

### 输入框
- 默认：白色背景，灰色边框
- Focus：蓝色边框
- Error：红色边框 + 错误文字
- Placeholder：浅灰色

### 按钮
- 主要按钮：蓝色背景，白色文字
- 次要按钮：白色背景，蓝色边框和文字
- Hover：背景色加深
- Disabled：灰色背景，禁止光标

### Toast提示
- 位置：页面右下角
- 动画：从右侧滑入，2秒后自动消失
- 类型：成功（绿色）、错误（红色）、警告（橙色）

## 6. 技术方案

### 技术栈
- 纯HTML5 + CSS3 + Vanilla JavaScript
- 无框架依赖，确保最快加载速度
- 使用QRCode.js库实现二维码生成（CDN引入）
- 所有工具纯客户端实现，无需后端服务器

### 文件结构
```
/
├── index.html          # 主页面（包含所有工具）
├── css/
│   └── style.css       # 所有样式
├── js/
│   └── app.js          # 所有工具逻辑
└── SPEC.md             # 项目规格文档
```

### SEO优化
- 每个工具独立的标题和Meta描述
- 语义化HTML标签
- Open Graph标签
- 结构化数据（JSON-LD）

### 性能目标
- 首屏加载：<1秒
- 工具响应：<100ms
- 无外部字体阻塞渲染
